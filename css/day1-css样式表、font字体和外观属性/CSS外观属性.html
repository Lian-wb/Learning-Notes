color：文本颜色 1、预定义颜色值：用颜色英文表示 2、十六进制表示：#打头，后面跟6位数字 3、RGB代码 text-align：文本水平对齐方式 1、left：左对齐 2、right：右对齐 3、center：居中对齐 line-height：行间距（行高） 一般分三种：像素px，相对值em和百分比%，但通常用px 一般情况下，行距比字号大7、8像素比较合理 text-indent：首行缩进 其属性可为不同单位的数值，1em相当于一个字的宽度，或相对于浏览器窗口宽度的百分比%，允许使用负数 text-decoration：文本装饰 none：无装饰 blink：闪烁 underline：下划线 line-through：贯穿线 overline：上划线
